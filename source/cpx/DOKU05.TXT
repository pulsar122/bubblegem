- mehr RAM (dadurch keine Stackprobleme mehr)
- BGEM-Cookie (nur zur Laufzeit von BG vorhanden, d.h. wird beim
  Start von BG angelegt und bei AP_TERM entfernt):
    typedef struct
    {
      long   magic;   /* 'BGEM' */
      long   size;    /* Grîûe dieser Struktur, derzeit 16 */

                           0  sonst */

    } BGEM;
  Die Struktur liegt im globalen Speicher.
- Aufruf aus modalen Dialogen (s. allgemeine Routine in call.txt):
    * wie unmodal, aber BGS7_USRHIDE ($0001) in pipe[7] gesetzt
    * nach Mausauswertung durch aufrufende Applikation BUBBLEGEM_HIDE
      ($babf) an BG schicken

DÑmon:
- zeitgesteuerte Hilfe bei angepaûten Applikationen ganz einfach durch
  Starten des Hilfe-DÑmons (HLPDEMON.APP)
- Ausschalten durch Beenden des DÑmons (AP_TERM schicken, z.B. mit
  AppLine)
- Anpassung von Applikationen ist denkbar einfach:
    * beim Empfang von BUBBLEGEM_REQUEST ($baba) ruft man BubbleGEM
      ganz normal (allerdings mit den Koordinaten aus BUBBLEGEM_REQUEST)
      auf, nachdem man anhand der empfangenen Koordinaten festgestellt
      hat, ob man fÅr diese Position eine Hilfe anbieten kann
    * pipe[0]:=BUBBLEGEM_REQUEST;
      pipe[1]:=demonID;
      pipe[2]:=0;
      pipe[3]:=winID; { AES-ID des Fensters unter dem Mauscursor }
      pipe[4]:=mX;    { Koordinaten der Maus }
      pipe[5]:=mY;
      pipe[6]:=KStat; { Status der Umschalttasten }
      pipe[7]:=0;
- der DÑmon legt den Cookie BGHD an, mit den der DÑmon gesteuert werden
  kann (der Aufbau ist eigentlich nur fÅr Konfigurationsprogrammprogrammierer
  interessant):
    * oberes Word:  Zeit bis zur Anzeige der Hilfe in ms (Default 500)
    * unteres Word: Bitmap; mit Bit $0001 (BGHD_ACTIVE) kann der DÑmon an-
      (1) bzw. ausgeschaltet (0) werden
    * nach einer énderung des Cookies muû man den DÑmon durch Senden von
      BUBBLEGEM_REQUEST (msg[2..7]=0) zum Neueinlesen der Cookiewerte
      veranlassen
- die Zeit bis zur Anzeige kann auch per Kommandozeile ("-t anzahl_der_ms")
  bestimmt werden
